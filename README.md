## Usage Diagram

![Usage Diagram](https://github.com/R1013-T/slide-share/blob/main/assets/usage_diagram_slide_share_web.jpg)

## Usage

```bash
# Enable Corepack to manage your Node.js package managers
corepack enable

# Install dependencies with Bun
bun i

# Run dev with bun
bun dev
```
